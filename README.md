[Course website](https://www.kaggle.com/learn-guide/5-day-agents)

## Getting Started
1. Install requirements (sorry for using too large environment for now, I will later update to a cleaner environment):  
   ```bash
       pip install -r requirements.txt
   ```
2. Choose which day agents you want to try and change to its working directory:
    ```bash
       cd day2
   ```
3. Run web agent development kit: 
    ```bash
       adk web
   ```
4. Choose agent
5. Start chatting with it!
